## О приложении

Данное приложение представляет собой галарею уникальных котиков.
При каждом обновлении страницы появляются случайные котики из пула.
Реализована пагинация (вечная подрузка котиков) для бесконечного скролла галереи.
Есть возможность добавлять и удалять любимых котиков, просматривать их на отдельной странице.
Котики хранятся по следующей ссылке: https://thecatapi.com.

## Как запустить приложение
Для того, чтобы запустить приложение, установите все требуемые пакеты с помощью npm install.
После установки пакетов убедитесь, что в корне проекта появилась папка node_modules со всеми необходимыми пакетами.

После установки пакетов требуется создать .env-файл в корне проекта.
В .env-файле должна содержаться переменная API_KEY, ваш личный API-ключ, который вы можете получить по ссылке выше.

После создания .env-файла в терминале в корне проекта требуется прописать npm start.
После того, как вы пропишите npm start, проект начнет собираться и запустится сервер вместе с браузером.

## Технологии, используемые в проекте
Для сборки проекта используется webpack с настроенным webpack-конфигом. 
Webpack-конфиг находится в корневой папке проекта и его частные модули в config.

В проекте для типизации используется TypeScript. Для стилизации настроены scss-модули.

Для создания приложения используется React, react-router-dom для роутинга, async-react-helmet для СЕО-оптимизации.
Для контроля состояний используется Redux Toolkit и react-redux
Для запроса к удаленной API используется библиотека axios.

Любимые котики хранятся локально с помощью localStorage.

Для работы на разных устройствах реализованы media-запросы.
